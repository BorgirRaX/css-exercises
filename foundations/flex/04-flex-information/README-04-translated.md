# Fitur website yang sangat umum

Tujuan dari latihan ini adalah membuat ulang sebuah bagian yang ditemukan di banyak website informasional.

Untuk yang ini lo juga perlu mengedit HTML sedikit. Kita tidak bisa membuat semuanya _terlalu_ mudah untuk lo. Lo akan ingin menambahkan container di sekitar berbagai elemen agar lo bisa mengatur mereka dengan flex. Semangat!
