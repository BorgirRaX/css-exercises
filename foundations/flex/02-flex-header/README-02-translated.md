# Header Dasar

Gunakan aturan flexbox untuk membuat gaya header halaman web yang sangat umum ini. Keuntungan menggunakan flex di sini adalah semuanya harus _fleksibel_. Lihat dua screenshot di bawah untuk gambaran bagaimana header ini harus menyesuaikan dengan ukuran layar lo. Selain aturan flex, lo juga perlu menambahkan beberapa aturan untuk margin dan padding. (Petunjuk: `ul` punya margin/padding default yang perlu lo tangani.)

### Self Check
- Ada jarak antara semua item dan tepi header (jumlah px spesifik tidak masalah di sini).
- Logo dipusatkan secara vertikal dan horizontal.
- list-items bersifat horizontal, dan dipusatkan secara vertikal di dalam header.
- left-links dan right-links didorong sepenuhnya ke kiri dan kanan, dan tetap di tepi header ketika halaman diresize.
- Solusi lo tidak menggunakan floats, inline-block, atau absolute positioning.

- Catatan: Untuk latihan ini, tidak masalah jika font-family tidak cocok.
