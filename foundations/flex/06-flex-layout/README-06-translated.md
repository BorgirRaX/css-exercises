# Seluruh halaman!

Flexbox berguna untuk menyusun seluruh halaman serta komponen-komponen kecil yang sudah kita kerjakan sebelumnya. Untuk latihan ini, kita tinggalkan lo dengan sedikit lebih banyak pekerjaan yang harus dilakukan, dengan beberapa hal yang mungkin belum lo temui. Sangat wajar untuk googling hal-hal yang lo tidak yakin!

### Petunjuk
- Lo mungkin ingin mencari sesuatu seperti `CSS remove list bullets`. Kita sudah melakukan ini untuk lo di contoh-contoh sebelumnya, tapi tidak di sini. Yay learning.
- Mencari tahu cara menata link di CSS mungkin membantu lo menghilangkan dekorasi garis bawah yang menyebalkan itu...
- Kita sudah menambahkan `height: 100vh` ke `body`... ini membuat body tepat sama tingginya dengan viewport. Untuk menempelkan footer ke bawah lo perlu menggunakan flex dan mengubah direction menjadi column.

### Self Check

- Header ada di bagian atas halaman, footer di bagian bawah, dan mereka tetap di tempat jika lo resize layar.
- Header dan footer punya padding.
- Link di header dan footer didorong ke kedua sisi.
- Ada jarak antara link di header dan footer.
- Footer punya background abu-abu muda (`#eeeeee`).
- Logo, input dan button dipusatkan di layar.
- Button punya jumlah padding yang sesuai.
- Ada jarak antara logo, input dan button.
