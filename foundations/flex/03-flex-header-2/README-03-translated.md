# Gaya header umum lainnya

Kita mulai menyelipkan sedikit lebih banyak CSS yang belum lo lihat. Jangan khawatir tentang ini untuk sekarang; kita hanya ingin semuanya terlihat sedikit lebih cantik, dan CSS ini tidak akan mengganggu tugas lo.

Untuk yang ini lo mungkin perlu mengedit HTML sedikit. Seringkali dengan flexbox lo perlu menambahkan container di sekitar elemen agar mereka bisa pergi ke tempat yang lo perlukan. Dalam kasus ini, lo mungkin ingin memisahkan item yang ada di kiri dan kanan header.

Sama seperti yang lo lakukan di latihan `02-flex-header`, lo akan bersarang flex container di dalam satu sama lain.

## Hasil yang diinginkan
Seperti contoh terakhir, yang ini perlu fleksibel di tengah, dengan item didorong ke kiri dan kanan.

### Self Check
- Semuanya dipusatkan secara vertikal di dalam header.
- Ada jarak 8px antara semuanya dan tepi header.
- Item disusun secara horizontal seperti yang terlihat di gambar hasil.
- Ada jarak 16px antara setiap item di kedua sisi header.
- flex digunakan untuk mengatur semuanya.
